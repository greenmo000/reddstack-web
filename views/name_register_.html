Redd-ID is a decentralised, blockchain based protocol sharing social identity and providing a platform for micro-transactions and tipping functionality.

Redd-ID is maintained on a global network of computers, much the same as Reddcoin (or Bitcoin). Collectively the network holds and maintains a global registry of social identity and public keys on its own blockchain (sidechain). By running a node, you are not only participating but also securing the network against tampering and control (no central authority)

The Redd-ID global registry contains each owners key/value pairs that are associated for each social media identity. The registry can be queried to identfy and resolve recipients (akin to a global address book). Users of the registry can opt to make any of their information public or private.


  • Redd-ID provides:
    • user controlled idenities
    • selective publishing of identity information
    • user self administered
